<SPEC>
New File : 새로운 파일을 생성합니다.
Save File : 파일을 저장합니다. 제목이 없을 경우 제목을 정할 수 있습니다.
Save As : 다른 이름으로 파일을 저장합니다. 중복되는 이름이 있을 경우 저장할 수 없습니다.
Delete File : 해당 파일을 제거하고 탭을 닫습니다.

좌측의 파일 리스트 : 파일을 클릭하면 새로운 탭을 생성하여 내용을 불러옵니다. 같은 파일을 중복해서 열 수 없습니다.
                                 이미 파일이 열려있을 경우 해당 탭으로 이동합니다.

텍스트 편집기 상단의 탭 : 탭을 누르면 해당 파일을 편집할 수 있습니다. 
                                         탭을 더블클릭시 해당 탭을 닫습니다.
